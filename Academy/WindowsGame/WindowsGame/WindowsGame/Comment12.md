## 1. 씬매니저
```
 게임에서는 씬들이 여러개 존재한다.
 ex) 채널선택씬, 로비씬, 인게임씬 등 + 로딩씬
```

```
 씬매니저 구성요소

 활성화된 Scene을 가지고있을거다.
 활성화된 Scene에 Init, Render, Update 를 호출하도록 한다.

 그리고 씬 전환 함수가 호출되면
 현재 활성화된 Scene을 해제(Relase)하고
 전환될 씬을 Init하고
 Render, Update하도록 해준다.
```
