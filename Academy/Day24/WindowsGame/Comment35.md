## UI - Button

### 버튼의 기능
 - 눌렀을때 원하는 함수를 호출
 - 버튼 상태에 따라 다르게 이미지가 나와야함.
     - [Hover, Press, Normal, Disabled]
 (마우스커서가 버튼위로 올라가면 다른 이미지가 나오거나 색이 좀 변해야함.)


### 버튼
 - UI 일뿐.
 (UserInterface)
 - 버튼에서 핵심로직을 작성안하는걸 권장한다.
 
 ### 원하는 함수 실행.
  - 버튼안에서 핵심로직을 짜는건 비선호되는 내용이다.
  - 함수포인터, callback function 을 써야하는데

  ### 함수포인터 / Callback Function
